### Prerequisites
- **Bun** (for frontend and docs)
- **Deno** 2.0+ (for backend)
- **Git**

### Quick Setup
```bash
git clone https://github.com/code-forge-temple/agentic-signal.git
cd agentic-signal
bun install
cd client && bun install && cd ..
cd docs && bun install && cd ..
bun run dev
```
